## 3. 리스트 테스트
### 순서 없는 리스트
* 항목 A
* 항목 B
  * 하위 항목 B-1
  * 하위 항목 B-2
- 항목 C

### 순서 있는 리스트
1. 첫 번째
2. 두 번째
3. 세 번째

### 작업 리스트 (Task List)
- [x] 완료된 작업
- [ ] 진행 중인 작업
- [ ] 대기 중인 작업
